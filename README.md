# 🏦 Bank System Backend

A scalable backend system simulating real-world banking operations like transactions, authentication, and ledger management.

---

## 🚀 Features

* User Authentication (JWT)
* Account Creation & Management
* Secure Money Transfers
* Transaction History
* Ledger System

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose

---

## 📦 Installation

```bash
git clone https://github.com/arpankarak/bank-system-backend.git
cd bank-system-backend
npm install
npm start
```

---

## 🔑 API Endpoints

* POST /register
* POST /login
* POST /transfer
* GET /transactions
